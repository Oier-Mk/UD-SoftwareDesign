Our translation service is based on Google Translator. 
Language ID's are based on ISO-639-1.
You can check all supported languages at: https://cloud.google.com/translate/docs/languages. 
If a different language ID is used, the service will return an error.

Popular languages:
Spanish = es
English = en
French = fr
German = de
Swedish = sv
Italian = it

ATTENTION - Ant tasks and Eclipse, together, are non-interactive; you must use the console /terminal to run the client.
Execution steps:
1. ant build
2. ant client
